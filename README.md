# 💻 Android Studio를 이용한 손글씨 인식

## 📌포트폴리오 설명
###### "오픈소스프로젝트" 학교 전공수업에서 
###### 안드로이드스튜디오를 사용하여 직접 숫자 손글씨를 작성해
###### 인식 기능 작동이 잘되는지 실습했습니다.

## 📌포트폴리오 결과
###### 손글씨가 굵고 커야하며 완벽하게 특정숫자가 인식되지 않는 문제가 있었습니다.
